# CRM system

CRM system for managing users, products, orders, and more.

## Installation

### Prerequisites

To get started with CRM, ensure you have the following prerequisites installed and set up:

- Node.js 20.x or later (recommended to use Node.js 22.x or later)

### Getting Started

1. Install dependencies:

   ```bash
   npm install
   # or
   yarn install
   ```

   > Use the `--legacy-peer-deps` flag if you encounter peer-deps error during installation.

2. Start the development server:

   ```bash
   npm start
   # or
   yarn start
   ```

   Runs the app in the development mode.\
   Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
